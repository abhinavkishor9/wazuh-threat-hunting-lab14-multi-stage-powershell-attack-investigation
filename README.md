# wazuh-threat-hunting-lab14-multi-stage-powershell-attack-investigation
SOC investigation of a multi-stage PowerShell attack using Sysmon and Wazuh SIEM. This lab demonstrates event correlation across Process Creation (Event ID 1), Network Connection (Event ID 3), and File Creation (Event ID 11) to reconstruct an attack timeline and map activity to MITRE ATT&amp;CK.
